Para subir cambios al repositorio es necesario entrar a la carpeta en la que se encuentra desde la terminal y realizar los siguientes comandos:
cd desktop/G5-ED-Martha
1. git status (revisar nuevas modificaciones)
2. git add . (agregar todos los archivos nuevos)
3. git commit -m "descripción commit" (poner una descripción de los cambios realizados)
4. git push (subir los cambios)
